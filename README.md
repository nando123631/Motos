El sistema funciona como un contrato inteligente que define una serie de instrucciones (funciones) que los usuarios pueden ejecutar. Cada tienda está asociada a un propietario identificado por su clave pública, y dentro de ella se almacenan productos representados como motos, con atributos como marca, precio y disponibilidad.
El flujo comienza con la creación de una tienda, donde el propietario inicializa una cuenta en la blockchain con su marca y un inventario vacío. A partir de ahí, el dueño puede agregar motos, eliminarlas, alternar su disponibilidad (por ejemplo, marcar una moto como vendida o disponible), visualizar la lista completa de productos y consultar el número total de artículos registrados.
